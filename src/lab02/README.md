# ЛР-2 — Коллекция объектов

## Предметная область
Игровая логика (Game Logic)

## Объект
**Player** — модель игрового персонажа.

Более детально: [LAB01](../lab01/README.md)

## Коллекция
**PlayerCollection** — контейнер объектов Player.

**Возможности:**
* добавление и удаление объектов
* поиск по никнейму
* итерация
* индексация
* сортировка
* фильтрация

## Реализовано
* `add` / `remove` / `remove_at`
* `find_by_nickname`
* `__len__`, `__iter__`, `__getitem__`
* `sort_by_level`
* `get_alive`

### 1. Базовая работа с коллекцией
Добавление персонажей в отряд, проверка на дубликаты никнеймов и удаление элементов.
