# ЛР-1 — Сущность (Entity)

## Предметная область
Игровая логика (Game Logic)

## Объект
**Player** — модель игрового персонажа. 
Это базовая сущность системы, обладающая состоянием (здоровье, уровень, опыт) и поведением (получение урона, прокачка).

## Возможности
* Создание персонажа с валидацией данных.
* Система накопления опыта и автоматического повышения уровня.
* Обработка получения урона и проверка жизненного статуса.
* Строгая инкапсуляция полей через свойства (properties).
* Сравнение объектов по никнейму и классу.

## Реализовано
* `take_damage` / `gain_experience` — основные методы поведения.
* `is_alive` — проверка состояния объекта.
* `nickname`, `health`, `level` — геттеры и сеттеры с валидацией.
* `__str__`, `__repr__`, `__eq__` — магические методы для работы с объектом.
